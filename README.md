# Утро После — памятка при похмелье

Красивая, тёмная, мобильная HTML-страница-памятка: что делать, что есть и о чём стоит знать на утро после алкоголя.

**[👉 Открыть страницу](https://lisofoxa-oss.github.io/utro-posle)**

## Что внутри

- **🌅 Утро** — первая помощь: вода, бульон, бананы, яичница. Что нельзя (кофе натощак, «опохмел»).
- **☀️ День** — восстановление: лёгкая еда, режим, имбирный чай, рассол, ибупрофен.
- **⚠️ Цифры о вреде** — статистика ВОЗ, канцероген группы 1, влияние на сон и микробиом, «hangxiety».
- **👩 Для девушек** — почему женский организм уязвимее: ферменты, рак груди, гормоны, кожа, сердце.
- **🧭 Чек-лист** — короткие правила на утро.

Никакой морали — только факты и конкретные рекомендации.

## Для кого

Для всех, но особенно для молодых девушек — раздел с физиологическими особенностями и доказательной базой.

## Технически

- Чистый HTML + CSS, без зависимостей
- Mobile-first дизайн (100dvh, safe-area, `clamp()`)
- Адаптивная сетка: стопка на телефоне, 2–3 колонки на планшете/десктопе
- Sticky-навигация по разделам
- Тёмная тема
