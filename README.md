# BAC Calculator / Калькулятор опьянения

**Рассчитай уровень алкоголя в крови, узнай последствия и получи план восстановления.**  
**Calculate your blood alcohol level, check the effects and get a personalized recovery plan.**

🌐 **Live:** https://lisofoxa-oss.github.io/utro-posle/

---

## Что умеет / Features

- **BAC-расчёт** по формуле Видмарка (пол, вес, время)
- **60+ напитков**: пиво, вино, крепкое, ликёры, коктейли (Мохито, Маргарита, Негрони и др.)
- **Добавление нескольких напитков** — выбирай, вводи объём, добавляй ещё
- **Реалтайм-дашборд**: текущий BAC, стадия опьянения, время до 0‰, пик BAC
- **Персональный план восстановления** — что пить, есть, избегать
- **Памятка при похмелье** — коротко о главном
- **Предупреждения** при опасном уровне (0.15‰+)
- **Отдельный блок для девушек** — физиология женского организма
- **Русский / English** — переключается, сохраняется в localStorage

## Технически / Tech

- Чистый HTML + CSS + JS, без зависимостей
- Mobile-first, тёмная тема
- Анимации, sticky-навигация, модальные окна
- BAC: Widmark formula с учётом метаболизма (0.015‰/ч)

## Для кого

Для всех, кто хочет понимать свою меру — без морализаторства, по науке.  
For anyone who wants to understand their limits — no judgment, just science.
